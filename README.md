# Skin Cancer Classifier

Deep learning model for skin cancer classification using Python and PyTorch.

## Overview
This project builds a convolutional neural network (CNN) to classify dermatoscopic images of skin lesions using the HAM10000 dataset.

## Technologies
- Python
- PyTorch
- NumPy
- Matplotlib
- Jupyter Notebook

## Features
- Image preprocessing and normalization
- CNN model training
- Model evaluation and accuracy tracking

## Dataset
HAM10000 skin lesion dataset.

## Goal
The goal of this project is to apply deep learning techniques to assist in early detection of skin cancer using medical images.
